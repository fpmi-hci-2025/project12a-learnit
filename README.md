### Project Name:
LearnIt

### Description:
Проект предназначен для изучения и развития навыков программирования.

Имеются следующие функции: 
- регистрация/авторизация уникального пользователся
- список материалов в формате лекций 
- викторина с вопросами на основе лекции
- задача на языке программирования на каждую тему
- отслеживание прогресса 
- календарь с отображением дней, когда пользователь изучал материалы

Installation: Последовательность шагов, как установить приложение локально.
Sub modules: Ссылки на репозитории для веб-приложения, мобильного приложе-
ния с их кратким описанием.
Usage: Рекомендации как использовать приложение после установки. Может со-
держать скриншоты.
Contributing: Сведения об авторах проекта и какие задачи реализовывали.
